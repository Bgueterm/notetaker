# README

Ruby version is "ruby 2.6.0p0 (2018-12-25 revision 66547) [x86_64-linux]"

Rails version is "5.2.2.1"

To streamline running the app, it is best run in AWS Cloud9.

To run the rails server, navigate to the notetaker directory in terminal and execute the line: rails s

To view the server, navigate to "Preview" at the
top of the Cloud9 environment, click preview running application, then click the "open in external window" button in
the new window that is formed in the Cloud9 environment.
